# B 任务书 — 防区入侵检测（模块二，25 分核心）

> 角色定位：分值最高的核心模块。几何判定与时空防抖骨架已在 `intrusion.py` 写好，你负责接入检测模型与前后端联调。
> 关联设计：系统设计说明书 §5。

## 一、职责范围

1. **YOLOv8n 人员检测**
   - 加载 YOLOv8n 权重，输出人员检测框。
   - 以检测框底边中心点 `(cx, cy)` 作为判定基准点（§5.3，已实现于 `IntrusionDetector.base_point`）。
2. **时空碰撞判定（已有骨架，需接入）**
   - 复用 `detectors/intrusion.py` 的 `judge()`：`cv2.pointPolygonTest` 距离判定 + 无间断计时。
   - 包装成 A 的 `Detector` 插件（实现 `detect(frame, ts)`），注册进推理引擎，**不要自建线程池**。
3. **防区参数持久化（§5.2）**
   - `api/regions` 增删改查落库（与 D 的数据层对接）。
   - 处理前端 F 提交的多边形坐标：归一化坐标 → 原始分辨率映射。
4. **联调**
   - 与 F 联调画区交互，确认 polygon / `X_distance` / `Y_stay_time` 传参一致。

## 二、交付物

| 交付 | 说明 |
| --- | --- |
| 入侵检测插件 | 实现 A 的 `Detector` 接口，触发时产出告警事件（结构见 E） |
| regions 接口 | 防区 CRUD + 坐标映射 |
| 联调通过 | 画区 → 实时判定 → 告警 全链路 |

## 三、判定逻辑（已实现，务必遵守）

```
D = cv2.pointPolygonTest(polygon, (cx, cy), True)
① D >= 0                      → 闯入      → 计时器累计
② D < 0 且 |D| <= X_distance  → 低于安全距离 → 计时器累计
③ 其余                        → 安全      → 计时器清零
危险无间断累计 >= Y_stay_time  → 触发告警
```

## 四、验收标准

- [ ] 人员进入防区且停留超时正确告警。
- [ ] 抖动（短暂进出）不误报（计时清零生效）。
- [ ] 前端画区参数与后端判定一致，多分辨率不错位。
- [ ] 单元测试 `tests/test_intrusion.py` 全绿。

## 五、协作接口

- **上游**：A（帧 + Detector 接口）、F（画区参数）。
- **下游**：E（告警事件）、D（数据落库）。
